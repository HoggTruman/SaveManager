# Save Manager (Work In Progress)
A savefile manager for video games that use a single savefile such as the Dark Souls series.

## Features
* Register any game and create profiles to organize your savefiles
* Backup or load a game's savefile with the click of a button
* An intuitive UI to add, rename and delete savefiles and arrange them into folders
* (TO BE ADDED) Global hotkey support

## Technology Used
* C# / .NET 8.0
* [WPF](https://learn.microsoft.com/en-us/dotnet/desktop/wpf/overview/?view=netdesktop-8.0)
* [xUnit](https://xunit.net/) + [Moq](https://www.nuget.org/packages/Moq/)
* MVVM Architecture
* IO / Filesystem
